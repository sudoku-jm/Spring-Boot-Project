#Spring Boot Project


스프링부트 활용해 게시판 만들기 + 그 외 추가 기능
- 회원가입 , 회원권한
- 마이페이지 , 프로필, 회원정보 쓰기,수정
- 일반,갤러리,응용 게시판 보기,쓰기,수정,삭제기능
- 게시판 검색
- 게시판 페이징
- 댓글



## 사용기술
본 스프링 부트 게시판 만들기에 사용된 기술
- JAVA
- JAVASCRIPT
- Jquery
- HTML
- CSS
